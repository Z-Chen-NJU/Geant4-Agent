\subsection*{3. Interpretation}
\begin{itemize}
\item The \texttt{v3\_24} set is suitable as the current external demonstration baseline.
\item The \texttt{v4\_24} set is a material-colloquial stress test. It should be presented as an explicit robustness challenge, not as a solved benchmark.
\item The current gap is concentrated in colloquial material canonicalization and one source-type mismatch case, not in the orchestrator or overwrite control flow.
\end{itemize}
