\begin{itemize}
\item \texttt{v3\_24} as the main baseline
\item \texttt{v4\_24} as the robustness companion set
\end{itemize}
\subsection*{2. Current Recommendation}
\begin{itemize}
\item Use \texttt{v3\_24} as the primary external demo evidence.
\item Use \texttt{v4\_24} immediately after \texttt{v3\_24} to demonstrate colloquial-material grounding.
\item Present \texttt{v4\_24} as a stress-style companion set that has now passed full live regression.
\end{itemize}

\subsection*{4. What Changed in v4}
The \texttt{v4\_24} dataset replaces explicit Geant4 material tags in user utterances with colloquial material names, for example:
\begin{itemize}
\item \texttt{G4\_Fe} $\rightarrow$ \texttt{?} / \texttt{iron}
\item \texttt{G4\_Cu} $\rightarrow$ \texttt{?} / \texttt{copper}
\item \texttt{G4\_WATER} $\rightarrow$ \texttt{?} / \texttt{water}
\item \texttt{G4\_CsI} $\rightarrow$ \texttt{???} / \texttt{cesium iodide}
\end{itemize}
The expected final configuration still uses canonical Geant4 labels.
\subsection*{5. Interpretation}
\begin{itemize}
\item \texttt{v3\_24} remains the main external baseline.
\item \texttt{v4\_24} now also closes at \texttt{24/24} in live regression.
\item \texttt{v4\_24} should be shown as a robustness companion set because it is stricter than \texttt{v3\_24}.
\end{itemize}
